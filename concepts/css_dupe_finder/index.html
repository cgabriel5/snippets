cssdupefinder.js
CSSDupeFinder is a tool that scans CSS to find duplicate CSS declaration properties within a CSS code block.
How It Works
Say the follwing CSS code on the left. After running CSSDupeFinder the tool will return only code block(s) that contain duplicate declaration properties.
.my-class {
    background: red;
    padding: 10px;
    border-radius: 3px;
    background: lightgreen;
    display: flex;
}

html {
    box-sizing: border-box;
}

*,
:before,
:after {
    box-sizing: inherit;
}

p.description#sidebar + p:first-line,
[attribute$="value"], .post img:nth-of-type(n+2):nth-last-of-type(n+2) {
    content: "Figure number " counter(image) ": ";
    counter-increment: image;
    content: "some\" \#12;3";
}

img {
    opacity: 0.5;
    filter: alpha(opacity=50); /* For IE8 and earlier */
}
